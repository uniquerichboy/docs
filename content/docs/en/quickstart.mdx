## Step 1: Create an Account

Sign up at [my.soxway.io](https://my.soxway.io) and add funds to your balance.

## Step 2: Get an API Token



```bash cURL
curl -X POST https://api.soxway.io/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "email": "you@example.com",
    "password": "your-password",
    "token_name": "my-app"
  }'
```

```python Python
import requests

response = requests.post("https://api.soxway.io/v1/auth/token", json={
    "email": "you@example.com",
    "password": "your-password",
    "token_name": "my-app"
})

token = response.json()["data"]["token"]
print(f"Token: {token}")
```

```javascript Node.js
const response = await fetch("https://api.soxway.io/v1/auth/token", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    email: "you@example.com",
    password: "your-password",
    token_name: "my-app"
  })
});

const { data } = await response.json();
console.log(`Token: ${data.token}`);
```



Save the `token` from the response. You'll use it for all API calls.

## Step 3: Create a Proxy



```bash cURL
curl -X POST https://api.soxway.io/v1/proxies \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "residential",
    "country": "US",
    "behavior": "rotate",
    "rotate_type": "each_request"
  }'
```

```python Python
import requests

headers = {"Authorization": "Bearer YOUR_TOKEN"}

response = requests.post("https://api.soxway.io/v1/proxies", 
    headers=headers,
    json={
        "type": "residential",
        "country": "US",
        "behavior": "rotate",
        "rotate_type": "each_request"
    }
)

proxy = response.json()["data"]
print(f"HTTP:   {proxy['proxy_http']}")
print(f"SOCKS5: {proxy['proxy_socks5']}")
```

```javascript Node.js
const response = await fetch("https://api.soxway.io/v1/proxies", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    type: "residential",
    country: "US",
    behavior: "rotate",
    rotate_type: "each_request"
  })
});

const { data } = await response.json();
console.log(`HTTP: ${data.proxy_http}`);
console.log(`SOCKS5: ${data.proxy_socks5}`);
```



## Step 4: Use the Proxy

The response includes ready-to-use proxy URLs:



```bash cURL
# HTTP proxy
curl -x http://user:pass@proxy.soxway.io:8080 https://httpbin.org/ip

# SOCKS5 proxy
curl -x socks5://user:pass@proxy.soxway.io:1080 https://httpbin.org/ip
```

```python Python
import requests

proxies = {
    "http": "http://user:pass@proxy.soxway.io:8080",
    "https": "http://user:pass@proxy.soxway.io:8080"
}

response = requests.get("https://httpbin.org/ip", proxies=proxies)
print(response.json())
```

```javascript Node.js
import { HttpsProxyAgent } from 'https-proxy-agent';

const agent = new HttpsProxyAgent('http://user:pass@proxy.soxway.io:8080');

const response = await fetch('https://httpbin.org/ip', { agent });
const data = await response.json();
console.log(data);
```
