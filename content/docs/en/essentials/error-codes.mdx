All API errors follow a consistent format:

```json
{
  "error": {
    "code": 1001,
    "message": "Authentication token is missing.",
    "description": "Include a Bearer token in the Authorization header.",
    "field": null,
    "errors": null
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `code` | integer | Unique error code (see tables below) |
| `message` | string | Human-readable error message |
| `description` | string | Actionable description of how to fix the error |
| `field` | string \| null | The field that caused the error (validation only) |
| `errors` | array \| null | Detailed validation errors (validation only) |

---

## Authentication & Authorization (1xxx)

| Code | Name | HTTP Status | Description |
|------|------|-------------|-------------|
| 1001 | `MISSING_AUTH_TOKEN` | 401 | No Bearer token in Authorization header |
| 1002 | `INVALID_CREDENTIALS` | 401 | Wrong email or password |
| 1003 | `EXPIRED_TOKEN` | 401 | Token has expired |
| 1004 | `INVALID_TOKEN` | 401 | Token is malformed or revoked |
| 1005 | `INSUFFICIENT_PERMISSIONS` | 403 | Account lacks permission for this action |
| 1006 | `TWO_FACTOR_REQUIRED` | 403 | 2FA must be enabled and verified |
| 1007 | `RESELLER_NOT_ENABLED` | 403 | Reseller access is not enabled on this account |

## Validation (2xxx)

| Code | Name | HTTP Status | Description |
|------|------|-------------|-------------|
| 2001 | `VALIDATION_FAILED` | 422 | One or more fields failed validation |
| 2002 | `MISSING_REQUIRED_FIELD` | 422 | A required field is missing |
| 2003 | `INVALID_VALUE` | 422 | Field value is not in the allowed set |
| 2004 | `INVALID_FORMAT` | 422 | Field format is incorrect |
| 2005 | `VALUE_TOO_LONG` | 422 | Field exceeds maximum length |
| 2006 | `VALUE_TOO_SHORT` | 422 | Field is below minimum length |
| 2007 | `VALUE_OUT_OF_RANGE` | 422 | Numeric value is outside allowed range |
| 2008 | `DUPLICATE_VALUE` | 422 | Value already exists (must be unique) |
| 2009 | `INVALID_COUNTRY_CODE` | 422 | Country code is not a valid ISO 3166-1 alpha-2 code |
| 2010 | `INVALID_IP_FORMAT` | 422 | IP address or CIDR notation is malformed |
| 2011 | `INVALID_DOMAIN_PATTERN` | 422 | Domain pattern is malformed |
| 2012 | `BULK_LIMIT_EXCEEDED` | 422 | Too many items in bulk operation (max 500) |
| 2013 | `INVALID_PROXY_TYPE` | 422 | Proxy type is not valid |

## Not Found (3xxx)

| Code | Name | HTTP Status | Description |
|------|------|-------------|-------------|
| 3001 | `NOT_FOUND` | 404 | Generic resource not found |
| 3002 | `PROXY_NOT_FOUND` | 404 | Proxy configuration does not exist |
| 3003 | `CONFIGURATION_NOT_FOUND` | 404 | Configuration does not exist |
| 3004 | `TOKEN_NOT_FOUND` | 404 | Authentication token not found |
| 3005 | `SUB_USER_NOT_FOUND` | 404 | Sub-user does not exist or doesn't belong to this reseller |
| 3006 | `PRODUCT_NOT_FOUND` | 404 | Product or service type not found |
| 3007 | `BLOCKLIST_ENTRY_NOT_FOUND` | 404 | Blocked host entry not found |
| 3008 | `WHITELIST_ENTRY_NOT_FOUND` | 404 | IP whitelist entry not found |

## Business Logic (4xxx)

| Code | Name | HTTP Status | Description |
|------|------|-------------|-------------|
| 4001 | `INSUFFICIENT_BALANCE` | 402 | Account balance is too low for this operation |
| 4002 | `PAYMENT_GATEWAY_UNAVAILABLE` | 400 | Payment processing is temporarily unavailable |
| 4003 | `PROMO_CODE_INVALID` | 400 | Promo code is invalid or expired |
| 4004 | `PROMO_CODE_ALREADY_USED` | 400 | Promo code has already been redeemed by this account |
| 4005 | `PRODUCT_NOT_AVAILABLE` | 400 | Product or service type is currently disabled |

## Server & External (5xxx)

| Code | Name | HTTP Status | Description |
|------|------|-------------|-------------|
| 5001 | `SERVICE_UNAVAILABLE` | 503 | Service is temporarily unavailable |
| 5002 | `GATEWAY_UNAVAILABLE` | 503 | Proxy gateway is temporarily unavailable |
| 5003 | `INTERNAL_ERROR` | 500 | Unexpected server error |

## Handling Errors



```python Python
import requests

response = requests.get("https://api.soxway.io/v1/proxies", 
    headers={"Authorization": "Bearer YOUR_TOKEN"})

if response.status_code != 200:
    error = response.json()["error"]
    
    if error["code"] == 1001:
        print("Token missing â€” re-authenticate")
    elif error["code"] == 1004:
        print("Token invalid â€” generate a new one")
    elif error["code"] >= 5000:
        print("Server error â€” retry after a delay")
    else:
        print(f"Error {error['code']}: {error['message']}")
```
