## Key Features



  - **Geo Targeting**:
    Target by country, region, or city across 195+ countries
  
  - **Rotation Control**:
    Keep sticky sessions or rotate on each request or interval
  
  - **HTTP & SOCKS5**:
    Both protocols supported on every proxy
  
  - **Pay As You Go**:
    Only pay for the traffic you use, no subscriptions
  
  - **IP Whitelist**:
    Restrict access to your trusted IPs
  
  - **Domain Blocklist**:
    Block unwanted domains at the proxy level

## Authentication

The API uses Bearer token authentication. Get your token by calling the [Issue Token](/api-reference/auth/issue-token) endpoint with your email and password.

```bash
curl -X POST https://api.soxway.io/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "password": "your-password", "token_name": "my-app"}'
```

Then use the token in all subsequent requests:

```bash
curl https://api.soxway.io/v1/proxies \
  -H "Authorization: Bearer YOUR_TOKEN"
```
