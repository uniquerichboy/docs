Все ошибки API имеют единый формат:

```json
{
  "error": {
    "code": 1001,
    "message": "Authentication token is missing.",
    "description": "Include a Bearer token in the Authorization header.",
    "field": null,
    "errors": null
  }
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `code` | integer | Уникальный код ошибки |
| `message` | string | Человекочитаемое сообщение |
| `description` | string | Описание с инструкцией по исправлению |
| `field` | string \| null | Поле, вызвавшее ошибку (только для валидации) |
| `errors` | array \| null | Детальные ошибки валидации |

---

## Аутентификация и авторизация (1xxx)

| Код | Название | HTTP | Описание |
|-----|----------|------|----------|
| 1001 | `MISSING_AUTH_TOKEN` | 401 | Нет Bearer-токена в заголовке Authorization |
| 1002 | `INVALID_CREDENTIALS` | 401 | Неверный email или пароль |
| 1003 | `EXPIRED_TOKEN` | 401 | Токен истек |
| 1004 | `INVALID_TOKEN` | 401 | Токен повреждён или отозван |
| 1005 | `INSUFFICIENT_PERMISSIONS` | 403 | Нет прав для этого действия |
| 1006 | `TWO_FACTOR_REQUIRED` | 403 | Требуется включить и подтвердить 2FA |
| 1007 | `RESELLER_NOT_ENABLED` | 403 | Ресселерский доступ не активирован |

## Валидация (2xxx)

| Код | Название | HTTP | Описание |
|-----|----------|------|----------|
| 2001 | `VALIDATION_FAILED` | 422 | Одно или несколько полей не прошли валидацию |
| 2002 | `MISSING_REQUIRED_FIELD` | 422 | Обязательное поле отсутствует |
| 2003 | `INVALID_VALUE` | 422 | Значение не входит в допустимый набор |
| 2004 | `INVALID_FORMAT` | 422 | Неверный формат поля |
| 2005 | `VALUE_TOO_LONG` | 422 | Превышена максимальная длина |
| 2006 | `VALUE_TOO_SHORT` | 422 | Меньше минимальной длины |
| 2007 | `VALUE_OUT_OF_RANGE` | 422 | Числовое значение вне допустимого диапазона |
| 2008 | `DUPLICATE_VALUE` | 422 | Значение уже существует |
| 2009 | `INVALID_COUNTRY_CODE` | 422 | Невалидный код страны ISO 3166-1 alpha-2 |
| 2010 | `INVALID_IP_FORMAT` | 422 | Неверный формат IP-адреса или CIDR |
| 2011 | `INVALID_DOMAIN_PATTERN` | 422 | Неверный формат доменного паттерна |
| 2012 | `BULK_LIMIT_EXCEEDED` | 422 | Превышен лимит массовой операции (макс. 500) |
| 2013 | `INVALID_PROXY_TYPE` | 422 | Невалидный тип прокси |

## Не найдено (3xxx)

| Код | Название | HTTP | Описание |
|-----|----------|------|----------|
| 3001 | `NOT_FOUND` | 404 | Ресурс не найден |
| 3002 | `PROXY_NOT_FOUND` | 404 | Конфигурация прокси не существует |
| 3003 | `CONFIGURATION_NOT_FOUND` | 404 | Конфигурация не существует |
| 3004 | `TOKEN_NOT_FOUND` | 404 | Токен аутентификации не найден |
| 3005 | `SUB_USER_NOT_FOUND` | 404 | Суб-пользователь не найден |
| 3006 | `PRODUCT_NOT_FOUND` | 404 | Продукт или тип сервиса не найден |
| 3007 | `BLOCKLIST_ENTRY_NOT_FOUND` | 404 | Запись блоклиста не найдена |
| 3008 | `WHITELIST_ENTRY_NOT_FOUND` | 404 | Запись вайтлиста не найдена |

## Бизнес-логика (4xxx)

| Код | Название | HTTP | Описание |
|-----|----------|------|----------|
| 4001 | `INSUFFICIENT_BALANCE` | 402 | Недостаточно средств на балансе |
| 4002 | `PAYMENT_GATEWAY_UNAVAILABLE` | 400 | Платёжная система временно недоступна |
| 4003 | `PROMO_CODE_INVALID` | 400 | Промокод невалиден или истёк |
| 4004 | `PROMO_CODE_ALREADY_USED` | 400 | Промокод уже использован этим аккаунтом |
| 4005 | `PRODUCT_NOT_AVAILABLE` | 400 | Продукт или тип сервиса отключён |

## Сервер и внешние сервисы (5xxx)

| Код | Название | HTTP | Описание |
|-----|----------|------|----------|
| 5001 | `SERVICE_UNAVAILABLE` | 503 | Сервис временно недоступен |
| 5002 | `GATEWAY_UNAVAILABLE` | 503 | Прокси-шлюз временно недоступен |
| 5003 | `INTERNAL_ERROR` | 500 | Непредвиденная ошибка сервера |

## Обработка ошибок



```python Python
import requests

response = requests.get("https://api.soxway.io/v1/proxies",
    headers={"Authorization": "Bearer ВАШ_ТОКЕН"})

if response.status_code != 200:
    error = response.json()["error"]

    if error["code"] == 1001:
        print("Токен отсутствует — пройдите аутентификацию")
    elif error["code"] == 1004:
        print("Токен невалиден — сгенерируйте новый")
    elif error["code"] >= 5000:
        print("Ошибка сервера — повторите позже")
    else:
        print(f"Ошибка {error['code']}: {error['message']}")
```

```javascript Node.js
const response = await fetch("https://api.soxway.io/v1/proxies", {
  headers: { Authorization: "Bearer ВАШ_ТОКЕН" }
});

if (!response.ok) {
  const { error } = await response.json();

  if (error.code === 1001) {
    console.log("Токен отсутствует — пройдите аутентификацию");
  } else if (error.code === 1004) {
    console.log("Токен невалиден — сгенерируйте новый");
  } else if (error.code >= 5000) {
    console.log("Ошибка сервера — повторите позже");
  } else {
    console.log(`Ошибка ${error.code}: ${error.message}`);
  }
}
```
