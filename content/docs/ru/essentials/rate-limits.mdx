Все эндпоинты API ограничены по количеству запросов для обеспечения стабильной работы.

## Уровни лимитов



  #### Публичный API

    | Окно | Лимит | Область |
    |------|-------|---------|
    | В минуту | 60 запросов | На пользователя (или на IP) |
    | В час | 1 000 запросов | На пользователя (или на IP) |
    | В сутки | 10 000 запросов | На пользователя (или на IP) |
  

  #### Аутентификация

    | Окно | Лимит | Область |
    |------|-------|---------|
    | В минуту | 5 запросов | На IP-адрес |

    > **Warning:**
      Эндпоинт получения токена (`POST /v1/auth/token`) имеет строгий лимит для защиты от перебора.
    
  

  #### Гео-эндпоинты

    | Окно | Лимит | Область |
    |------|-------|---------|
    | В минуту | 30 запросов | На IP-адрес |

    Гео-эндпоинты (страны, регионы, города) не требуют аутентификации.
  

  #### Reseller API

    | Окно | Лимит | Область |
    |------|-------|---------|
    | В минуту | 120 запросов | На ID ресселера |
    | В час | 3 000 запросов | На ID ресселера |

    > **Info:**Ресселеры имеют повышенные лимиты для управления множеством суб-пользователей.
  




## Заголовки лимитов

| Заголовок | Описание |
|-----------|----------|
| `X-RateLimit-Limit` | Максимум запросов в текущем окне |
| `X-RateLimit-Remaining` | Оставшиеся запросы в текущем окне |
| `Retry-After` | Секунды до повтора (при ответе 429) |

## Обработка лимитов

При превышении лимита API возвращает HTTP `429 Too Many Requests`:

```json
{
  "error": {
    "code": 429,
    "message": "Too Many Requests",
    "description": "Rate limit exceeded. Please retry after 30 seconds."
  }
}
```

### Рекомендации



  #### Экспоненциальный откат

    При получении 429 подождите указанное в `Retry-After` время, затем повторите с увеличивающимися задержками.
  

  #### Кэшируйте гео-данные

    Страны, регионы и города меняются редко. Кэшируйте ответы локально и обновляйте периодически.
  

  #### Используйте массовые операции

    Для блоклиста используйте bulk-эндпоинт для добавления до 500 записей за один запрос.
  

  #### Мониторьте использование

    Проверяйте заголовок `X-RateLimit-Remaining` чтобы отслеживать расход до достижения лимита.
