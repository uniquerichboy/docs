## Создание прокси

Каждый прокси в Soxway — это **конфигурация**, определяющая тип, расположение и поведение вашего прокси-соединения.

### Типы прокси

| Тип | Описание | Применение |
|-----|----------|------------|
| `residential` | Реальные резидентные IP от ISP | Веб-скрапинг, маркетинговые исследования |
| `mobile` | IP мобильных операторов (3G/4G/5G) | Тестирование мобильных приложений, соцсети |
| `datacenter` | Быстрые IP дата-центров | Высокоскоростные задачи, массовые операции |
| `residential_premium` | Премиум резидентный пул | Высокий процент успеха, меньше блокировок |
| `mobile_premium` | Премиум мобильный пул | Лучшие мобильные IP |

### Создание через API

```bash
curl -X POST https://api.soxway.io/v1/proxies \
  -H "Authorization: Bearer ВАШ_ТОКЕН" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Мой US прокси",
    "type": "residential",
    "country": "US",
    "region": "California",
    "city": "Los Angeles",
    "behavior": "rotate",
    "rotate_type": "each_request"
  }'
```

### Создание через личный кабинет

1. Перейдите на [my.soxway.io/proxies/create](https://my.soxway.io/proxies/create)
2. Выберите тип прокси
3. Укажите страну, регион и город
4. Настройте поведение ротации
5. Нажмите **Создать прокси**

## Использование прокси

После создания вы получите данные для подключения:

| Поле | Пример |
|------|--------|
| `host` | `proxy.soxway.io` |
| `port_http` | `8080` |
| `port_socks5` | `1080` |
| `proxy_http` | `http://user:pass@proxy.soxway.io:8080` |
| `proxy_socks5` | `socks5://user:pass@proxy.soxway.io:1080` |

### Использование с популярными инструментами



  #### cURL

    ```bash
    # HTTP
    curl -x http://user:pass@proxy.soxway.io:8080 https://example.com

    # SOCKS5
    curl -x socks5://user:pass@proxy.soxway.io:1080 https://example.com
    ```
  

  #### Python (requests)

    ```python
    import requests

    proxies = {
        "http": "http://user:pass@proxy.soxway.io:8080",
        "https": "http://user:pass@proxy.soxway.io:8080"
    }

    response = requests.get("https://example.com", proxies=proxies)
    ```
  

  #### Node.js

    ```javascript
    import { HttpsProxyAgent } from 'https-proxy-agent';

    const agent = new HttpsProxyAgent('http://user:pass@proxy.soxway.io:8080');
    const response = await fetch('https://example.com', { agent });
    ```
  

  #### Браузер (Puppeteer)

    ```javascript
    const browser = await puppeteer.launch({
      args: ['--proxy-server=http://proxy.soxway.io:8080']
    });
    ```
  




## Управление прокси

### Список всех прокси

```bash
curl https://api.soxway.io/v1/proxies \
  -H "Authorization: Bearer ВАШ_ТОКЕН"
```

### Обновление прокси

```bash
curl -X PUT https://api.soxway.io/v1/proxies/\{id\} \
  -H "Authorization: Bearer ВАШ_ТОКЕН" \
  -H "Content-Type: application/json" \
  -d '{"country": "DE", "behavior": "keep"}'
```

### Удаление прокси

```bash
curl -X DELETE https://api.soxway.io/v1/proxies/\{id\} \
  -H "Authorization: Bearer ВАШ_ТОКЕН"
```
