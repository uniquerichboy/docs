---

Soxway поддерживает точный гео-таргетинг на трёх уровнях: **страна**, **регион** и **город**.

## Доступные локации

Используйте Geo API для получения списка доступных локаций. Эти эндпоинты **публичные** и не требуют аутентификации.

### Список стран

```bash
curl "https://api.soxway.io/v1/geo/countries?type=residential"
```

### Список регионов

```bash
curl "https://api.soxway.io/v1/geo/regions?country=US&type=residential"
```

### Список городов

```bash
curl "https://api.soxway.io/v1/geo/cities?country=US&type=residential"
```

> **Note:**
  Доступные локации различаются по типам прокси. Мобильные прокси могут иметь меньше городов, чем резидентные.


## Уровни таргетинга

### Только страна

```json
{ "type": "residential", "country": "US", "behavior": "rotate" }
```

### Страна + Регион

```json
{ "type": "residential", "country": "US", "region": "California", "behavior": "rotate" }
```

### Страна + Город

```json
{ "type": "residential", "country": "US", "region": "California", "city": "Los Angeles", "behavior": "rotate" }
```

> **Warning:**
  Поле `country` использует коды **ISO 3166-1 alpha-2** (например, `US`, `DE`, `GB`).


## Советы



  #### Кэшируйте гео-данные

    Страны, регионы и города меняются редко. Кэшируйте ответы и обновляйте раз в сутки.
  

  #### Начинайте с широкого таргетинга

    Если не нужна точность до города — таргетируйте по стране для лучшей доступности IP.
  

  #### Проверяйте доступность по типу

    Передавайте параметр `type` чтобы видеть локации, доступные для вашего типа прокси.
