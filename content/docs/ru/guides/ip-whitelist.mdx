---
title: 'IP Whitelist'
description: 'Ограничение доступа к прокси по IP-адресам'
---

IP-вайтлист позволяет ограничить доступ к вашим прокси только доверенным IP-адресам.

## Как это работает

Когда вы добавляете IP в вайтлист, только трафик с этих адресов будет проходить через ваши прокси. Все остальные соединения отклоняются.

> **Info:**
  Если вайтлист пуст, доступ к прокси контролируется только логином/паролем.


## Добавление IP

```bash
curl -X POST https://api.soxway.io/v1/account/whitelist \
  -H "Authorization: Bearer ВАШ_ТОКЕН" \
  -H "Content-Type: application/json" \
  -d '{
    "ip_or_cidr": "203.0.113.42",
    "comment": "Офисный сервер"
  }'
```

### CIDR-нотация

Можно добавить целую подсеть:

```json
{ "ip_or_cidr": "203.0.113.0/24", "comment": "Офисная сеть" }
```

## Список IP

```bash
curl https://api.soxway.io/v1/account/whitelist \
  -H "Authorization: Bearer ВАШ_ТОКЕН"
```

## Удаление IP

```bash
curl -X DELETE https://api.soxway.io/v1/account/whitelist/1 \
  -H "Authorization: Bearer ВАШ_ТОКЕН"
```

> **Warning:**
  Будьте осторожны при изменении вайтлиста. Удаление IP, с которого вы подключаетесь, немедленно заблокирует ваш доступ.
