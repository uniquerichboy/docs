## Что такое Soxway?

Soxway — это премиальный прокси-сервис, предоставляющий резидентные, мобильные и дата-центр прокси с точным гео-таргетингом вплоть до уровня города. Управляйте всем через личный кабинет или интегрируйте напрямую через наш REST API.



  - **[Быстрый старт](/quickstart):
    Запустите первый прокси за 5 минут
  
  - **[API Справочник](/api-reference/auth/issue-token):
    Полная документация API с интерактивным playground
  
  - **[Reseller API](/api-reference/reseller/overview):
    Создайте свой прокси-бизнес с нашей ресселерской программой
  
  - **[Гайды](/guides/proxy-setup):
    Пошаговые руководства для типичных задач
  



## Ключевые возможности



  - **Гео-таргетинг**:
    Таргетинг по стране, региону или городу в 195+ странах
  
  - **Управление ротацией**:
    Стики-сессии или ротация на каждый запрос / по интервалу
  
  - **HTTP и SOCKS5**:
    Оба протокола поддерживаются на каждом прокси
  
  - **Pay As You Go**:
    Платите только за использованный трафик, без подписок
  
  - **IP Whitelist**:
    Ограничьте доступ только доверенным IP-адресам
  
  - **Блокировка доменов**:
    Блокируйте нежелательные домены на уровне прокси
  



## Базовый URL

Все API-запросы отправляются на:

```
https://api.soxway.io
```

## Аутентификация

API использует Bearer-токен аутентификацию. Получите токен, вызвав эндпоинт [Получить токен](/api-reference/auth/issue-token) с вашим email и паролем.

```bash
curl -X POST https://api.soxway.io/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "password": "ваш-пароль", "token_name": "my-app"}'
```

Затем используйте токен во всех последующих запросах:

```bash
curl https://api.soxway.io/v1/proxies \
  -H "Authorization: Bearer ВАШ_ТОКЕН"
```
